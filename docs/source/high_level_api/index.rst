High-level Python API
=====================

The :py:mod:`vc2_bit_widths` Python module exposes the underlying functionality
provided by the command-line utilities (see :ref:`cli`) via a Python interface.
In addition, Python functions are provided for assisting in the serialisation
and deserialisation of the file formats used by the command-line utilities.
